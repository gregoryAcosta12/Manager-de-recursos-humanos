## 📖 Descripción del Proyecto

**HR Manager** es una aplicación web de gestión de recursos humanos diseñada para optimizar la administración de empleados en empresas de cualquier tamaño. Este sistema proporciona una interfaz intuitiva y profesional que permite a los equipos de RRHH gestionar eficientemente toda la información relacionada con el personal.

Actualmente el proyecto se encuentra en **fase de desarrollo activo**, con un enfoque en construir una base sólida y escalable que pueda extenderse con nuevas funcionalidades en el futuro.

### 🎯 Estado Actual del Proyecto

✅ **Completado:**
- Arquitectura base del sistema
- Dashboard con estadísticas principales
- CRUD completo de empleados
- Persistencia de datos con LocalStorage
- Diseño responsivo con Tailwind CSS
- Componentes reutilizables
- Sistema de filtros y búsqueda

🚧 **En Desarrollo:**
- Perfil detallado de empleado
- Gestión de vacaciones
- Sistema de asistencia
- Módulo de reportes avanzados

## 🚀 Tecnologías Utilizadas

### **Frontend**
- **[React 18.3](https://reactjs.org/)** - Librería principal para la construcción de interfaces de usuario
- **[TypeScript 5.5](https://www.typescriptlang.org/)** - Tipado estático para un código más robusto y mantenible
- **[Vite 5.3](https://vitejs.dev/)** - Build tool ultrarrápido para desarrollo moderno
- **[Tailwind CSS 4.0](https://tailwindcss.com/)** - Framework CSS utilitario para estilos personalizados
- **[Lucide React](https://lucide.dev/)** - Iconos modernos y consistentes

### **Características Técnicas**
- **Context API** - Gestión de estado global sin dependencias externas
- **LocalStorage** - Persistencia de datos en el navegador
- **Custom Hooks** - Lógica reutilizable y limpia
- **Componentes Funcionales** - Arquitectura moderna con Hooks
- **CSS Animations** - Transiciones y animaciones suaves con Tailwind
